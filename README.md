# pyTodo
This is a Todo application available from the CLI.<br>
## use

### install...
```
$ git clone https://github.com/gamma-410/pyTodo.git  
$ cd pyTodo  
$ pip install -e .  
```

### Commands
Add:  
```$ todo add [value]```

Show:  
```$ todo show```  

Delete:  
```$ todo delete [listNum]```

Clear:  
```$ todo clear```
